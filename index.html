insights Impact

Web masters published front page content by manually changing the HTML code. This made the process error prone and so I created a tool that generated the code automatically from a set of clickable options. Resulted in an increased similar task completion speed by at least 500% and reduced error rate by 99%.

Web masters used too many confusing and outdated publishing flows for any time a business area requested something to be published. Thus I developed an engine where the business areas across the bank uploaded their own format type of content (pdf,xls,docx,svg etc) and it would be transformed automatically into a web friendly format (mostly HTML) with the meta-data recognized and assigned to be published to the proper section as scheduled. This resulted in allowing roughly 5000 colleagues with zero web technical knowledge the ability to publish content on the website handling the conversions automatically.

I created a design tool so that our bank colleagues would be able to generate infographic content for social media while keeping the ECB theme and style intact. This reduced the workload for the design team by at least 50% for social media requests since people were creating what they needed on their own.

I had been so enthusiastic about the projects that I was rubbing off my excitement to people radiating happiness and joy across the whole division thus fostering an uplifting enviroment for everybody closeby. Resulted in increased team performance and job satisfaction.
